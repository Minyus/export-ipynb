# nbexport, Python script to export .ipynb file from .py

This is a Python script to export .ipynb file from .py using `#%%` as the separator of cells.
This tool is useful to run Python scripts in Jupyter or Google Colab (with GPU) keeping the code chunks separated as cells.

This Python script can be registered as an external tool in PyCharm as shown in the following screenshot.
